# WhatsUsea

Application qui récupére les observations de certains animaux marins à proximité des îlots de la province sud de la Nouvelle-Calédonie.

## Pré-requis

* Node.js
* Yarn
* Vue CLI
* Python
* PIP

## Structure

| Location             |  Content       |
|----------------------|----------------|
| `/backend`           | Django Project |
| `/frontend`          | Vue App        |

## Cloner le projet

```shell
git clone WhatsUsea.git
cd WhatsUsea
```
